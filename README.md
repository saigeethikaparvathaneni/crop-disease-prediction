# 🌱 Crop Disease Prediction using Machine Learning

## 📌 Overview
This project focuses on predicting crop diseases using environmental and soil parameters such as temperature, humidity, rainfall, soil type, and pH levels.

The goal is to enable early detection of plant diseases, helping farmers take preventive measures and improve crop yield.

---

## 🚀 Features
- Predicts crop diseases using Machine Learning
- Uses weather and soil data for analysis
- Helps in early disease detection
- Improves agricultural decision-making
